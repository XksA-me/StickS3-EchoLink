# README 图片资源

当前 SVG 是公开仓库首页的占位图，替换真实图片时建议：

| 内容 | 建议文件 | 建议尺寸 |
| --- | --- | --- |
| 项目 Banner | `banner.png` | 2350 x 1000，比例 2.35:1 |
| 公众号“简说 Python”二维码 | `wechat-official.png` | 800 x 800 或更高 |
| 个人微信二维码 | `wechat-personal.png` | 800 x 800 或更高 |

加入 PNG 后，将根目录 `README.md` 中的占位 SVG 路径替换为对应 PNG 路径。二维码周围保留足够白边，并在 GitHub 桌面端和手机端各扫描一次确认。
